⚽ Шаг 2. Рисуем мяч и заставляем его двигаться

✏️ Твоя задача:

Напиши код, который:

  1. Получает элемент canvas и контекст для рисования.
  2. Рисует мяч (круг радиусом 8-10 пикселей).
  3. Заставляет мяч двигаться по экрану после нажатия кнопки «Старт».
  4. Очищает холст перед каждым кадром.
  5. Реализует отскок от стен (левой, правой и верхней).
📖 Подробное объяснение (читай внимательно):
  • canvas и контекст: const canvas = document.getElementById('gameCanvas'); — находим наш холст. const ctx = canvas.getContext('2d'); — получаем инструмент для рисования (как кисть).
  • Переменные мяча: x, y — координаты центра мяча. dx, dy — скорость движения по горизонтали и вертикали (пикселей за кадр).
  • Функция drawBall(): ctx.beginPath() — начинаем рисовать фигуру. ctx.arc(x, y, radius, 0, Math.PI*2) — рисуем круг. ctx.fill() — заливаем цветом.
  • Функция draw(): ctx.clearRect(0,0,width,height) — стирает всё с холста. Потом рисуем мяч. Если игра запущена, обновляем координаты мяча и проверяем отскоки от стен.
  • Отскок от левой/правой стены: Если x + dx выходит за границы холста (меньше радиуса или больше width-радиус), меняем dx = -dx (скорость становится противоположной).
  • Отскок от верхней стены: Если y + dy меньше радиуса, меняем dy = -dy.
  • Анимация: requestAnimationFrame(draw) — просит браузер перерисовать следующий кадр (60 раз в секунду). Благодаря этому мяч движется плавно.
  • Кнопка старт: По нажатию устанавливаем gameRunning = true, чтобы мяч начал двигаться.
🔑 Посмотреть полный код решения с пояснениями 
// 1. Получаем элементы
const canvas = document.getElementById('gameCanvas');
const ctx = canvas.getContext('2d');

// 2. Переменные для управления игрой
let gameRunning = false;   // игра не запущена

// 3. Параметры мяча
let x = canvas.width / 2;    // центр по горизонтали
let y = canvas.height - 30;  // центр по вертикали (чуть выше низа)
let dx = 2;                  // скорость вправо
let dy = -2;                 // скорость вверх
const ballRadius = 8;        // радиус мяча

// 4. Функция рисования мяча
function drawBall() {
    ctx.beginPath();                 // начинаем новую фигуру
    ctx.arc(x, y, ballRadius, 0, Math.PI * 2);  // круг
    ctx.fillStyle = "#0095DD";       // цвет мяча
    ctx.fill();                      // заливаем
    ctx.closePath();                 // завершаем фигуру
}

// 5. Главная функция рисования (вызывается каждый кадр)
function draw() {
    ctx.clearRect(0, 0, canvas.width, canvas.height); // очищаем экран
    drawBall();                                        // рисуем мяч
    
    if (gameRunning) {                                 // если игра активна
        // Отскок от левой и правой стен
        if (x + dx > canvas.width - ballRadius || x + dx < ballRadius) {
            dx = -dx;   // разворачиваем горизонтальную скорость
        }
        // Отскок от верхней стены
        if (y + dy < ballRadius) {
            dy = -dy;   // разворачиваем вертикальную скорость
        }
        
        // Обновляем координаты мяча
        x += dx;
        y += dy;
    }
    
    requestAnimationFrame(draw);  // просим следующий кадр
}

// Запускаем бесконечную анимацию
draw();

// 6. Кнопка "Начать игру"
document.getElementById('startBtn').addEventListener('click', () => {
    gameRunning = true;
});

🏓 Шаг 3. Добавляем ракетку и управление

✏️ Твоя задача:

Добавь в игру ракетку, которой можно управлять:

  1. Ракетка должна быть прямоугольником шириной 75px и высотой 10px.
  2. Ракетка должна двигаться влево и вправо с помощью стрелок клавиатуры только когда игра запущена.
  3. Ракетка не должна выходить за границы canvas.
  4. Реализуй отскок мяча от ракетки.
  5. Добавь обработчик события mousemove для управления мышью.
📖 Подробное объяснение:
  • Ракетка: paddleWidth — ширина, paddleHeight — высота. paddleX — её x-координата (левый верхний угол).
  • Управление клавишами: Создаём флаги rightPressed и leftPressed. При нажатии стрелки флаг становится true, при отпускании — false. В каждом кадре, если игра запущена, двигаем ракетку на 7 пикселей в нужную сторону, но не выходим за пределы холста.
  • Управление мышью: canvas.addEventListener('mousemove', ...) — получаем координату мыши относительно canvas. paddleX = mouseX - paddleWidth/2 — ставим ракетку под курсор. Проверяем границы, чтобы ракетка не вылезала за край.
  • Отскок от ракетки: Проверяем, когда мяч достигает высоты ракетки (низ холста минус высота ракетки). Условие: y + dy > canvas.height - ballRadius - paddleHeight — мяч коснулся верха ракетки. И x > paddleX && x < paddleX + paddleWidth — мяч находится над ракеткой. Если всё верно, меняем dy = -dy.
🔑 Посмотреть полный код решения с пояснениями 
// 1. Параметры ракетки
let paddleHeight = 10;
let paddleWidth = 75;
let paddleX = (canvas.width - paddleWidth) / 2;  // изначально по центру
let rightPressed = false;
let leftPressed = false;

// 2. Слушатели клавиатуры
document.addEventListener('keydown', (e) => {
    if (e.key === 'ArrowRight') rightPressed = true;
    if (e.key === 'ArrowLeft') leftPressed = true;
});
document.addEventListener('keyup', (e) => {
    if (e.key === 'ArrowRight') rightPressed = false;
    if (e.key === 'ArrowLeft') leftPressed = false;
});

// 3. Управление мышью
canvas.addEventListener('mousemove', (e) => {
    if (!gameRunning) return;               // двигаем только когда игра идёт
    const relativeX = e.clientX - canvas.offsetLeft; // координата мыши внутри canvas
    if (relativeX > 0 && relativeX < canvas.width) {
        paddleX = relativeX - paddleWidth / 2;  // центр ракетки под курсором
        // Не вылезаем за левый край
        if (paddleX < 0) paddleX = 0;
        // Не вылезаем за правый край
        if (paddleX > canvas.width - paddleWidth) paddleX = canvas.width - paddleWidth;
    }
});

// 4. Функция рисования ракетки
function drawPaddle() {
    ctx.beginPath();
    ctx.rect(paddleX, canvas.height - paddleHeight, paddleWidth, paddleHeight);
    ctx.fillStyle = "#0095DD";
    ctx.fill();
    ctx.closePath();
}

// 5. Дописываем в функцию draw() после проверки стен:
if (gameRunning) {
    // Движение ракетки по клавишам
    if (rightPressed && paddleX < canvas.width - paddleWidth) paddleX += 7;
    if (leftPressed && paddleX > 0) paddleX -= 7;
    
    // Отскок от ракетки
    if (y + dy > canvas.height - ballRadius - paddleHeight && 
        y + dy < canvas.height - ballRadius &&
        x > paddleX && x < paddleX + paddleWidth) {
        dy = -dy;
    }
}

// 6. Не забыть вызвать drawPaddle() внутри draw():
function draw() {
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    drawBall();
    drawPaddle();   // рисуем ракетку
    // ... остальное
}

🧱 Шаг 4. Создаём кирпичи

✏️ Твоя задача:

Добавь кирпичи, которые нужно разбивать мячом:

  1. Создай массив кирпичей (3 строки, 5 столбцов).
  2. Каждый кирпич должен иметь координаты x, y и статус (1 — целый, 0 — разбит).
  3. Напиши функцию drawBricks(), которая рисует только целые кирпичи.
  4. Добавь функцию initGame() для сброса кирпичей при перезапуске.
🔑 Посмотреть решение с пояснениями 
// 1. Константы кирпичей
const brickRowCount = 3;          // строк
const brickColumnCount = 5;       // столбцов
const brickWidth = 75;            // ширина
const brickHeight = 15;           // высота
const brickPadding = 10;          // отступ между кирпичами
const brickOffsetTop = 30;        // отступ сверху
const brickOffsetLeft = 30;       // отступ слева

let bricks = [];                  // массив для хранения кирпичей

// 2. Функция создания/сброса кирпичей
function initBricks() {
    bricks = [];
    for (let c = 0; c < brickColumnCount; c++) {
        bricks[c] = [];
        for (let r = 0; r < brickRowCount; r++) {
            bricks[c][r] = { x: 0, y: 0, status: 1 };
        }
    }
}

// 3. Рисуем кирпичи
function drawBricks() {
    for (let c = 0; c < brickColumnCount; c++) {
        for (let r = 0; r < brickRowCount; r++) {
            if (bricks[c][r].status === 1) {
                const brickX = c * (brickWidth + brickPadding) + brickOffsetLeft;
                const brickY = r * (brickHeight + brickPadding) + brickOffsetTop;
                bricks[c][r].x = brickX;
                bricks[c][r].y = brickY;
                ctx.beginPath();
                ctx.rect(brickX, brickY, brickWidth, brickHeight);
                ctx.fillStyle = "#0095DD";
                ctx.fill();
                ctx.closePath();
            }
        }
    }
}

// 4. В функции initGame() (при старте/перезапуске) вызываем initBricks()

💥 Шаг 5. Обнаружение столкновений

✏️ Твоя задача:

Реализуй столкновение мяча с кирпичами:

  1. Напиши функцию collisionDetection().
  2. При попадании меняй направление мяча и устанавливай статус кирпича в 0.
  3. Добавь переменную score для подсчёта очков.
  4. При победе останавливай игру и показывай сообщение.
🔑 Посмотреть решение 
let score = 0;

function drawScore() {
    ctx.font = "16px Arial";
    ctx.fillStyle = "#0095DD";
    ctx.fillText("Score: " + score, 8, 20);
}

function collisionDetection() {
    for (let c = 0; c < brickColumnCount; c++) {
        for (let r = 0; r < brickRowCount; r++) {
            const b = bricks[c][r];
            if (b.status === 1) {
                if (x > b.x && x < b.x + brickWidth && 
                    y > b.y && y < b.y + brickHeight) {
                    dy = -dy;
                    b.status = 0;
                    score++;
                    if (score === brickRowCount * brickColumnCount) {
                        gameRunning = false;
                        updateStatusMessage("🎉 ПОБЕДА! 🎉");
                    }
                }
            }
        }
    }
}

  2. При потере жизни — уменьшай счётчик и сбрасывай позицию мяча и ракетки.
  3. При Game Over — останавливай игру и показывай сообщение.
  4. Реализуй кнопку «Начать заново» — полный сброс игры.
🔑 Посмотреть решение 
let lives = 3;

function drawLives() {
    ctx.font = "16px Arial";
    ctx.fillStyle = "#0095DD";
    ctx.fillText("Lives: " + lives, canvas.width - 65, 20);
}

function resetBallAndPaddle() {
    x = canvas.width / 2;
    y = canvas.height - 30;
    dx = 2;
    dy = -2;
    paddleX = (canvas.width - paddleWidth) / 2;
}

// В функции draw() добавляем проверку потери жизни:
if (y + dy > canvas.height - ballRadius) {
    lives--;
    if (lives === 0) {
        gameRunning = false;
        updateStatusMessage("💀 GAME OVER 💀 Очки: " + score);
    } else {
        resetBallAndPaddle();
        updateStatusMessage("❤️ Жизнь потеряна! Осталось: " + lives);
    }
}

// Функция полного перезапуска
function restartGame() {
    gameRunning = true;
    lives = 3;
    score = 0;
    initBricks();
    resetBallAndPaddle();
    updateStatusMessage("Игра перезапущена!");
}

document.getElementById('restartBtn').addEventListener('click', restartGame);
